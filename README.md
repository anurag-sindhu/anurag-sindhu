## 💡 Engineering Insights

**01 · gRPC Binary Efficiency** — Writing `1` as an integer costs 8 bits; as binary it needs just 1 bit — an 8× reduction. Why gRPC crushes REST+JSON at scale. `#gRPC`

**02 · UUID Primary Keys** — Using UUIDs as SQL primary keys eliminates sequential enumeration, preventing entire classes of **IDOR attacks** at the architecture level. `#Security`

**03 · Leaderless Replication** — Achieves strong consistency via quorum reads/writes. When **R + W > N**, overlap is guaranteed — no stale reads. `#DistributedSystems`

**04 · Bloom Filters** — Eliminate unnecessary DB calls trading just **1.13 MB** for a 1% false positive rate across 1 million records. `#Performance`

**05 · UUID Collision** — Generating 1 billion UUIDs/second for 100 years only brings duplicate probability to 50%. Ship with confidence. `#Reliability`
